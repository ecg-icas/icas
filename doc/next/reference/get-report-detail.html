GET /report/detailÂ¶

Scope

console_ro or api_ro

Accept

application/sellside.report.detail.list-v1+json, application/json

Warning

This call is scheduled to be deprecated. Please use the faster and synchronous replacement endpoint GET /metrics/ads instead. Keep in mind that the new endpoint only returns data for ads which have had performance-related activity in the requested period.

This URL returns the list of previously submitted reports for the current user. A maximum of 3 reports are returned.

ErrorsÂ¶

There are no specific errors for this call.
