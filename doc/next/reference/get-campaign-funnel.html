GET /campaign/{id}/funnel¶

Scope

api_ro or console_ro

Accept

application/sellside.campaign-targeting-v2+json, application/json

Returns a Campaign Funnel Object for the specified campaign. If the id is invalid, i.e., not a positive integer, the server returns 400 Bad Request. The funnel estimates the daily performance of the specified campaign, based on the the budgets, targeting settings and the currently-active ads in the campaign.

Campaign Funnel Object¶

{
    "buyerRanges": {
        "potential": {
            "min": <number>,
            "max": <number>
        },
        "total": {
            "max": <number>
        }
    },
    "visitorRanges": {
        "clicks": {
            "min": <number>,
            "max": <number>
        },
        "URLClicks": {
            "min": <number>,
            "max": <number>
        }
    }
}

Example:¶

{
    "buyerRanges": {
        "potential": {
            "min":3500,
            "max":4500
    },
    "total": {
            "max":15000
        }
    },
    "visitorRanges": {
        "clicks": {
            "min":550,
            "max":700
        },
    "URLClicks": {
            "min":220,
            "max":430
        }
    }
}
