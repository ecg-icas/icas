Campaigns¶

Campaigns allow the seller to group ads together, having shared budgets, same targeting rules, etc. Reporting can also be done on campaigns, instead of individual ads, as well as status management: It is possible to pause a campaign, resulting in all ads in the campaign to be taken offline. When a campaign reaches one of its budgets, all ads in that campaign will be taken offline. When budget becomes available again, all ads in the campaign will come online again.

Components¶

A campaign consists of several components:

Component Short description
Campaign id The unique reference to the Campaign
Campaign title User-provided title
Campaign vendorId Unique reference for the seller to the campaign
Campaign status Status of the campaign
Campaign links Ads in this campaign
Campaign budgets Budgets of this campaign

Campaign id¶

Unique reference to the Campaign, needs to be omitted in a POST or PUT action. Generated by us.

Campaign title¶

The title of the campaign, provided by the user. This makes it easier for people to know which campaign is which.

Campaign vendorId¶

A per-seller unique string to identify the campaign. This is provided by the seller. Can either be set when creating a campaign or when updating an existing campaign. However, once set, it can no longer be modified. When fetching an existing campaign which does not have a vendorId, the field is omitted.

Campaign status¶

The status the campaign currently has. Is from the list of Campaign Status. The user can set only one of the user controlled states ACTIVE, PAUSED or DELETED. DELETED is a final state.

Campaign links¶

A ‘Link’ defines which ad is in which campaign, and with which possible overrides for certain campaign properties(budgets, cpc, ...) This construction makes ads into mere data containers, and can allow for an ad to be part of multiple campaigns (future plans). A campaign without links is an empty campaign, which is a valid scenario.

Campaign budgets¶

Campaigns can have multiple types of budgets, see Campaign Budgets. Budgets are seen as limiting factors, meaning as soon as one runs out, all ads of this campaign will be taken offline until more budget becomes available.
