GET /feed/emtpy
================

.. list-table::
 :widths: 20 80

 * - Scope
   - <none>

 * - Accept
   - ``text/xml; charset=UTF-8``

This URL returns an empty file that can be used to temporarily pause the entire ad inventory. It provides a downloadable file with a unique
name, sufficient to upload it to your xml web server and change the URL to it.
