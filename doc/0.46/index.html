Introduction¶

This document describes the Admarkt Sellside API to place ads on Admarkt.

The API is designed as a RESTful API supporting JSON as data formats. Authentication is provided va OAuth2 to allow simple integration with existing client libraries.

If you have questions or would like to provide feedback you can send an email to Admarkt-API@marktplaats.nl.
