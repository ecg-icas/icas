GET /feed/xsd
================

.. list-table::
 :widths: 20 80

 * - Scope
   - <none>

 * - Accept
   - ``text/xml``

This URL returns the XSD used to validate **feed** XML files against. No scopes are needed for this call.
Users can use this XSD to check their **feed** XML themselves, instead of just submitting it and waiting for
results the next day.
